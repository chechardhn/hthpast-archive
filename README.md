# hthpast-archive

## 项目简介

hthpast-archive 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 资源的集合存储与展示空间。所有页面均为独立文件，不依赖外部数据库或后端服务。

## 目录结构

```
.
├── pages/              # 存放主要 HTML 页面文件
├── assets/             # 公共资源文件（CSS、JS、图片等）
├── index.html          # 仓库入口页面（可选）
└── README.md           # 本文件
```

- `pages/`：按时间或主题整理的 HTML 页面归档。
- `assets/`：页面引用的公共样式、脚本或媒体资源。

## 页面归档说明

- 每个 HTML 页面均为独立、自包含的静态文件。
- 页面内容可能涉及历史记录、技术演示、文档快照等，不保证时效性或准确性。
- 归档仅用于资料保存与展示，不代表对页面内容的认可或背书。
